Για την εκτέλεση του προγράμματος μας αρκεί να μεταγλωττιστούν τα αρχεία .java
Εκτός από την εισαγωγή τους σε οποιδήποτε περιβάλλον IDE σας εξυπηρετεί,ακολουθώντας τις
δικές του οδηγίες για μεταγλώττιση και εκτέλεση προγραμμάτων, μπορείτε να εκτελέσετε την 
εντολή javac *.java από την Γραμμή Εντολών των Windows, αφού πρώτα έχετε επιλέξει το σωστό 
directory που περιλαμβάνει τα αρχεία (cd path/3180141_3180150/src -> javac *.java). Στην 
συνέχεια απλά εκτελείτε την εντολή java Othello και ακολουθείτε τις οδηγίες που θα εμφανιστούν 
στην οθόνη σας. 

Καλή Διασκέδαση!

-----------------------------------------------------------------------------------------

In order to run our program, all .java files must be compiled. You can either import them into
any IDE enviroment of your choice, following its compilation and running guidelines, or execute
the javac command from your Windows Command Promt after first you have chosen the correct directory
that contains the files' folder (cd path/3180141_3180150/src -> javac *.java). Then just run the 
Othello file which contains the main method (java Othello) and follow the instructions on your screen.
